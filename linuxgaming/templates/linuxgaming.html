Graphical libraries

  • OpenGL
    • Develop by the Kronos Group
    • 2D and 3D graphical library
    • +25 years project
    • Graphical library to interact with GPU
  • Vulkan
    • Next generation OpenGL
    • Modern API
    • Better multiprocessor support
    • Language to manage shaders
    • Simplify cross platform development

Graphical libraries

Under the hat of Mesa 3D project

  • Mesa 3D
    • +25 years project
    • An open source implementation of graphics API
    • Gallium 3D (modern drivers) part of Mesa 2009
    • Not used by Nvidia who uses its own implementation
  • Api supported
    • OpenGL
    • OpenGL ES (embedded)
    • EGL (link with display manager)
    • GLX (link with display manager)
    • Vulkan
    • OpenCL (computing)
    • Direct 3D 9

  • Xorg
