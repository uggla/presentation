Graphical libraries

  • OpenGL
    • Develop by the Kronos Group
    • 2D and 3D graphical library
    • +25 years project
    • Graphical library to interact with GPU
  • Vulkan
    • Next generation OpenGL
    • Modern API
    • Better multiprocessor support
    • Language to manage shaders
    • Simplify cross platform development

Graphical libraries

Under the hat of Mesa 3D project

  • Mesa 3D
    • +25 years project
    • An open source implementation of graphics API
    • Gallium 3D (modern drivers) part of Mesa 2009
    • Not used by Nvidia who uses its own implementation

    • OpenGL

    • GLX (link with display manager)

What is the problem to play windows games

  • MFC
    • Library C++ that encapsulate Win32 API
  • Direct 3D (Direct X)
    • API 3D
    • OpenGL competitor

These libraries are not cross platform !

MFC and Direct 3D emulation

  • Wine
    • Compatibility layer to run win32 applications
  • DXVK
    • Vulkan-based translation layer for Direct3D 10/11 which allows running 3D applications on Linux using Wine
  • d9vk
    • Same as above but Direct3D 9 related
  • Proton
    • Compatibility tool for Steam Play based on Wine and additional components
